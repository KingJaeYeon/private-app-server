## 🌐 Global Errors (공통 발생 가능)

아래 오류들은 모든 API에서 공통적으로 발생할 수 있으며,  
서버 내부 처리 또는 시스템 정책에 의해 자동으로 반환될 수 있습니다.

### 🔧 시스템 오류
| Code | HTTP Status | Message |
|------|------------|---------|
| INTERNAL-001 | 500 | 일시적인 오류가 발생했습니다. 잠시 후 다시 시도해주세요. |
| INTERNAL-002 | 500 | 일시적인 오류가 발생했습니다. 잠시 후 다시 시도해주세요. |
| INTERNAL-003 | 500 | 일시적인 오류가 발생했습니다. 잠시 후 다시 시도해주세요. |
| INTERNAL-004 | 503 | 서비스를 일시적으로 이용할 수 없습니다. |
| INTERNAL-005 | 500 | 일시적인 오류가 발생했습니다. 잠시 후 다시 시도해주세요. |

### 🌐 글로벌 정책 오류
| Code | HTTP Status | Message |
|------|------------|---------|
| GLOBAL-001 | 429 | 요청이 너무 많습니다. 잠시 후 다시 시도해주세요. |
| GLOBAL-002 | 403 | 요청을 처리할 수 없습니다. 관리자에게 문의하세요. |
| GLOBAL-003 | 400 | 검증에 실패했습니다. 다시 입력해주세요. |

---

※ GLOBAL 에러는 개별 API 응답 스펙(@ApiResponse)에서 제외하고,  
**공통 에러 처리기(Global Exception Filter)**에서 자동 관리됩니다.
